## Server

#### Usage:
```bash
dnsex server [OPTIONS] --domain <DOMAIN>
```

#### Options:
  `-d, --domain <DOMAIN>`  
  `-b, --bind <BIND>`      [default: 0.0.0.0]
  `-p, --port <PORT>`      [default: 8053]
  `-h, --help`             Print help
